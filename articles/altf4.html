Then fear not! The AltF4++ is for you!

How to use it?

First, get the app from here



After that Extract the RAR to a empty folder, run "AltF4++.exe".

Now, you are gone see this ui.
( ! Importent note ! : the app will be not visible to avoid getting caught on the task bar, if it unfocuses use "ALT+TAB" to bring it up back.)


Now I am going to explain what does each elemented represented as a number do.

1. Element: You need to select the ".exe" file of the game you are going to play

2. Element: This is the most important part,this is the image that going to overlay the game, I recommand a picture of the desktop if you are playing it on full screen borderless.

3. Element: This is a save button (the save is file is going to be created in the same directory as the "AltF4++.exe" file), it saves the currently selected stuff.

4. Element: You need to select the save file that the 3. element created to load it.

5. Element: Choses the app that you want to open for eg: scratch (check elemnt 6 for context).

6. Element: Check the circle that suits you needs best, ("Site aç": Open website,"Uygulama seç": open an app) this is what it does after killing the game procces.

7. Element: This is the director that you have chrome is installed, dont have to change this one if you have chrome installed in the deafult directory.

8. Element: (check elemnt 6 for context) The url that the app will open after killing the game procces if "Site aç" is sellected

Now you know what every elemnt does I am going to tell you how to use the app.
After selecting all the necesery directorys to the image,the game etc you can hit alt+Z to overlay the image on the game hitting it again will close it and open the website/app you have selected for it. Here is a video example of it:
