I want to talk about some mechanics I can't realy decide where I should add them or not but first I will talk about the main issues the first game had.



So the biggest problem with the first game was people not understanding the goal of the game and how the controlls worked. I have addresed the issue with the controlls and prob will adress the difficulty in understanding the goal of the game soon.

Another problem was the lack of levels and difficulty balencing. As far as I recall there were 25 levels in the first game which would become quite repetetive for a longterm player (like if it had any :| ) and none of those levels were forgiving in terms of difficulty if the hard to understand controlls didn't get you the difficulty of the levels would sure stop you right on your tracks.

Rest of the problems were not that big but they are still worth a mention.
