Development (+Explination how it works)

The base of the site was realy easy to build ,code and do the css. The real time consuming part was to actualy write the "articles".

First I thought of creating a separete html page for each project obiusly this is not the correct thing to do so I did some thinking and dicaded to only have one html page that loads a bare html from a text file into a div. You are on that html page right now. Each article has an id for eg this articles id is "thisWeb" you can see it on the url bar on top. Here is a link to the barebone version of this article. And the "my projects" page works by looping thru a json which has small arrays that contaion info about the project (title, description etc.).

I also thinked about adding ads here, I even had how I would handle it planned (put an empty object with a specific in any point of an article then replace it with an ad iframe) ,but thought it wasn't worth the clutter + I hate ads.
